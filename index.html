HIC Prediction using Machine Learning

Project Abstract

I developed a CNN-based neural network for predicting Hi-C interactions from genomic sequences, enhancing the model with chromHMM protein annotations. The results show a significant improvement, indicating increased predictive power by incorporating chromatin states.

1. Introduction

Chromatin's 3D structure is vital for cellular functions, but linking linear genome sequences to spatial structures remains challenging. My work utilizes deep learning to predict Hi-C matrices from genomic sequences, offering a swift and cost-effective alternative to traditional methods.

1.1 Contributions

  • Implemented a PyTorch-based CNN architecture (Base-Model).
  • Extended the Base-Model to include ChromHMM sequence, demonstrating improved predictive ability (ChromHMM-Model).
  • Optimized the 1D to 2D transformation as a self-attention operation.

2. Background

Understanding genome folding is crucial, and Hi-C sequencing provides insights into chromosomal interactions. Denser Hi-C regions correlate with heterochromatin, while less dense regions associate with euchromatin, revealing the genome's functional state. My model focuses on predicting locus-specific genome folding from 1 Mb DNA sequences.

3. Approach

3.1 Data Preprocessing

  • Hi-C Matrices: Processed and interpolated missing values.
  • Genome Sequences: One-hot encoded for learnability.
  • ChromHMM Input: One-hot encoded from integer states.

3.2 Base-Model

  • Utilized 1D convolutional layers, ResNet layers, and a self-attention layer.
  • Trained on default MSE loss.

3.3 ChromHMM-Model

  • Added an extra 'head' to incorporate ChromHMM sequence.
  • Applied 1D convolutional layers, ResNet layers, and self-attention before combining ChromHMM with the genetic sequence.

3.4 Loss Function

Used mean squared error (MSE) loss and experimented with weighted MSE (W-MSE).

Solving Quantile Crossing for Prediction Interval of Forecast

Introduction

During my internship at Oracle, I worked on solving the problem of Quantile Crossing, a challenge in Quantile regression. This project aimed to address the limitations of existing models, specifically focusing on long seasonality data.

Background

Quantile Regression is a statistical technique used to model the conditional distribution of a response variable. Quantile Crossings occur when the predicted quantiles of a distribution intersect, leading to inaccurate predictions. In the context of forecasting, this can result in unreliable prediction intervals.

To tackle this issue, I devised an Incremental Quantile Functions (IQF) approach. Instead of predicting quantiles directly, I created an IQF block that predicts the distance between quantiles. By enforcing these distances to be only positive, I successfully eliminated Quantile Crossings.

Strategy

  • Implemented a Seq-2-Seq model using LSTM in encoder and decoder on a standard data set.
  • Trained the model on various loss functions such as pinball loss, multi-pinball loss, Continuous Rank
  • Probability Score, and penalized crossing to reduce the number of Quantile crossings.
  • Implemented approaches such as Incremental Quantile Functions (IQF) and Distributional Output instead of point estimation, thereby fully eliminating the Quantile Crossings.
  • Evaluated the results using Weighted Quantile loss (WQL), Mean Squared Error (MSE), Mean Absolute Error (MAE), Symmetric Mean Squared Error for each model.
  • Implemented SARIMA and SARIMAX using Statsmodel (a library in Python) on a self-generated dataset; Used Fourier Features of various lengths as exogenous features for SARIMAX.

Result

Evaluated the different model performances using Mean Squared Error on validation and test, finally recommending using IQF layer with multi-pinball loss to train the model. Recommended using SARIMAX with one Fourier Feature as it performed better than SARIMA.
